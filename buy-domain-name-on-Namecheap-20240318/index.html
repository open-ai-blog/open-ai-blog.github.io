2：选购域名

可以看到这个域名已经被购买了，那我们就需要选购其他的域名，一般.com的域名相对来说价格要高点，如果我们只是自己使用，没有其他特别需求的话，可以选购其他的域名，例如 .cn .top .vip的域名，这样的比较便宜。比如下面这个，我们选中之后，添加进购物车

接着就到了付款页面

我们看到上面的付款方式需要国外的银行卡，信用卡或者其他国外可支持的方式，如果我们有相关的支付方式或者我们有外国的朋友的话，直接填写相关的信息进行付款就可以了。如果没有的话，我们可以注册一个国外的虚拟信用卡 WildCard，使用邀请码 LAODENGAI 可以便宜2$（近15元人民币），打开 https://bewildcard.com/i/LAODENGAI 这个链接进行注册。（虚拟信用卡和实体信用卡一样，国外的支付场景基本上都可以用到，例如ChatGPT账户一键注册，ChatGPT Plus一键升级，如果没有魔法去登录ChatGPT使用的话，还有OpenAI专用的浏览器。另外还提供了一键申请海外的邮箱，进行收发邮件；还有提供海外的手机号，可以通过注册海外服务时候的手机短信验证，比如 OpenAI API Key 申请，Cluade 注册，Google 注册，Telegram 注册等。另外我们还可以通过 WildCard 去订阅Midjourney，OnlyFans，Sora等）

点击上面的链接进去后，直接点立即使用

再点击去注册

填写手机号，输入验证码（使用邀请码 LAODENGAI 在开卡时可以优惠2美元）

点击下一步进行支付宝扫码认证

支付宝认证完成后，选择开通时长（使用邀请码 LAODENGAI 可以便宜2美元，2年的是13.99美元，1年的是9.99美元）

点击确认支付，支付成功后就开卡成功了！然后点击我的服务页面，就可以看到我们的虚拟信用卡信息了。

上面的卡号，地址，安全码等信息都可以直接复制使用。

接着我们点虚拟卡上的充值按钮，选择线上充值

选择支付宝充值（充值有3.5%的手续费，最低充值10$，冲多也不用担心，可以提现，提现不用手续费。）

4：付款购买域名
