GPT账号注册太麻烦？如何一键注册GhatGPT账号！

|字数总计:1.4k|阅读时长:4分钟|阅读量:

从AlphaGo打败柯洁后，我们看到了AI的强大，然而这种专注于下围棋或者机器翻译的还只是小模型，GPT却是AI大模型，它更像我们人类的大脑。有理由相信随着AI技术的发展，ChatGPT也会发展到一个我们想象不到的高度，即时它可以帮我们做到我们能想到的几乎任何的事情！极大的增加我们的生产力（作为一个码农，不知道擅长crud的我，是不是要注定被淘汰了o(╥﹏╥)o)。

然而由于很多的原因，很多国内的用户没法直接使用ChatGPT，只能听别人说，自己心里痒痒的想去尝试，但是苦于没有账号，不过没关系，今天我就来教大家怎么注册官方的ChatGPT账号。

如果你想订阅升级ChatGPT4.0的话，可以查看这篇文章：GPT4.0 如何升级？使用 WildCard 虚拟信用卡开通升级 ChatGPT4.0 教程

废话不多说，下面我们就来注册官方ChatGPT账号！

1.如何注册海外邮箱

因为现在国内的邮箱都不允许注册OpenAI了，所以如果你已经拥有一个海外邮箱，那么你可以直接看第二步，注册OpenAI账户。但是请注意，如果你的邮箱是outlook，hotmail，live这类微软账户，基本上只要升级GPT4.0就会被封号！！！这个一定要注意！！已经有很多人被封号了。但是谷歌邮箱是可以的，毕竟是自家的嘛，下面我们就来看看谷歌邮箱怎么注册。

1.1 申请谷歌邮箱

首先我们进入谷歌创建账号页面 ，输入我们的姓名，年龄和性别等信息（可以随便填)

然后接着可以选择自动生成的地址或者自己创建一个邮箱地址，接着点击下一步输入密码

然后点击下一步，下拉选择印度尼西亚（+62）

1.2 注册WildCard

注册一个国外的虚拟信用卡 WildCard，使用邀请码 LAODENGAI 可以便宜2$（近15元人民币），打开 https://bewildcard.com/i/LAODENGAI 这个链接进行注册，注册完成后，WildCard会免费送我们3次申请外国手机号的机会。

为保留篇幅，此处仅保留文字步骤，详细的步骤可以看这篇文章：

WildCard 是什么？可以为我们提供什么服务？虚拟信用卡及 WildCard 使用教程

文字步骤如下（看着步骤多，其实很简单，跟着一步步点下去就行了）：

1.首先点击链接进入，WildCard一分钟注册，轻松订阅支付海外服务

2.进去后点立即使用

3.再点击去注册

4.接着填写我们自己的手机号，输入验证码（使用邀请码 LAODENGAI 在开卡时可以优惠2美元）

5.选择支付宝实名认证即可，方便快捷

6.然后填写你的个人信息（填写个人信息主要是平台为了防止xiqian，博主用这个平台已经半年多，很稳定）

7.点击下一步进行支付宝扫码认证

8.支付宝认证完成后，选择开通时长（2年的是13.99美元，1年的是9.99美元）

9.开通成功之后就可以使用虚拟信用卡来订阅服务了，直接在订阅的地方，把下面卡片的信息复制粘贴进去就行了

1.3 海外手机号验证谷歌邮箱

接着我们在WildCard网站点击我的服务，然后下拉找到海外手机号，点击申请手机号，选择谷歌，然后把获取到的手机号复制出来

接着回到我们的谷歌邮箱注册页面，把刚才申请到的手机号粘贴进去，点击下一步来到验证码页面

然后回到刚才注册的WildCard申请的手机号页面接收验证码即可

然后把这个验证码填入我们注册谷歌邮箱的页面即可。验证成功，我们的谷歌邮箱就注册好了。

2.申请OpenAI账号

首先打开OpenAI的注册页面 ，输入我们刚才申请的谷歌邮箱

输入邮箱密码后，成功发送邮件验证，直接点击页面打开邮箱，跳转到谷歌邮件页面，找到openai发送的验证邮件，点击进行验证即可

验证好后，我们回到OpenAI注册页面，补充完个人信息，点继续后，我们的OpenAI账户就注册好了。然后选择ChatGPT账户就行。

OpenAI 的用户体系分成两大类：ChatGPT 用户和API开发者用户，两个体系是相互独立的，用户名密码是一致的，但是访问入口和支付都是独立的：

ChatGPT 用户的登录入口是 https://chat.openai.com
API 开发者用户的登录入口是 https://platform.openai.com

ChatGPT 用户获取 GPT-4 的使用权限，是通过购买 GPT Plus 会员，会员的收费标准是 20 美元/每月。购买 GPT Plus 会员后，可以在 Chat 界面选择 GPT4 进行聊天。购买教程可以看这篇文章↓
