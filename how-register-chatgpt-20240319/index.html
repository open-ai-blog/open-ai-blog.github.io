那么该怎么注册GPT账号呢？

如果你想订阅升级ChatGPT4.0的话，可以查看这篇文章：GPT4.0 如何升级？使用 WildCard 虚拟信用卡开通升级 ChatGPT4.0 教程

废话不多说，下面我们就来注册官方ChatGPT账号！

1.注册海外邮箱

因为现在国内的邮箱都不允许注册OpenAI了，所以如果你已经拥有一个海外邮箱，那么你可以直接看第二步，申请OpenAI账户。但是请注意，如果你的邮箱是outlook，hotmail，live这类微软账户，基本上只要升级GPT4.0就会被封号！！！这个一定要注意！！已经有很多人被封号了。

在这里我推荐2个国外邮箱，一个是谷歌的gmail邮箱，这个需要手机号，可以用我们自己的手机号，不过为了以防万一，最好是用国外的手机号注册。另一个就是不需要手机号的proton邮箱，这个目前也是比较稳定的。

1.1 申请谷歌邮箱

具体的可以看这篇文章：如何注册谷歌邮箱 gmail

1.2 注册proton邮箱

具体的可以看这篇文章：不用手机号注册海外邮箱 Proton

2.申请OpenAI账号

首先打开OpenAI的注册页面 ，输入我们刚才申请的邮箱

输入邮箱密码后，成功发送邮件验证，直接点击页面打开邮箱，跳转到谷歌邮件页面，找到openai发送的验证邮件，点击进行验证即可

验证好后，我们回到OpenAI注册页面，补充完个人信息，点继续后，我们的OpenAI账户就注册好了。然后选择ChatGPT账户就行。

OpenAI 的用户体系分成两大类：ChatGPT 用户和API开发者用户，两个体系是相互独立的，用户名密码是一致的，但是访问入口和支付都是独立的：

ChatGPT 用户的登录入口是 https://chat.openai.com
API 开发者用户的登录入口是 https://platform.openai.com

ChatGPT 用户获取 GPT-4 的使用权限，是通过购买 GPT Plus 会员，会员的收费标准是 20 美元/每月。购买 GPT Plus 会员后，可以在 Chat 界面选择 GPT4 进行聊天。教程可以看这篇文章↓
